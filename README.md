# Requirements
To get this config working, you'll need:
* Latest version of Neovim (https://neovim.io)
* Nerd Fonts (https://nerdfonts.com)
* Latest version of Git (https://git-scm.com)

# Install
<details>
<summary>Unix</summary>
<br>

Create `nvim` directory:
```
cd ~/.config && mkdir ./nvim
```

Clone this repository to necessary directory:
```
git clone https://github.com/king-of-ducks/nvim.git ~/.config/nvim
```

Delete the `README.md` file (optionally)
```
rm -r ~/.config/nvim/README.md
```
</details>

<details>
<summary>Windows (PowerShell)</summary>
<br>

Move to Local AppData:
```
cd %localappdata%
```
Create `nvim` folder:
```
md nvim
```

Clone this repository to necessary directory:
```
git clone https://github.com/king-of-ducks/nvim.git %localappdata%/nvim
```
</details>
